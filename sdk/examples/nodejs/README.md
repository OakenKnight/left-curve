### Using the NodeJS Example

1. **Install dependencies**:
    ```sh
    pnpm install
    ```

2. **Run the example**:
    ```sh
    pnpm x <filename>
    ```

    Replace `<filename>` with the actual file you want to execute, such as `execute.ts` or `getChainInfo.ts`.

For more details, refer to the [HACKING.md](../../HACKING.md)
